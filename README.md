# Promedios Escolares

Este proyecto es una colección de ejercicios y mini-aplicaciones en JavaScript, HTML y CSS para calcular promedios, trabajar con bucles y generar tablas dinámicas.  
Ideal para estudiantes y practicantes de programación que quieran aprender lógica básica.

## 🚀 Funciones principales
- Cálculo de promedios de notas escolares.
- Ejemplos con bucles: `for`, `while`, `do...while`.
- Tablas dinámicas y estructura de boletín escolar.
- Interfaz simple y fácil de usar.

## 🖥️ Tecnologías usadas
- **HTML5** para la estructura.
- **CSS3** para estilos.
- **JavaScript** para la lógica y la interacción.
